# 快捷 emoji😀

这是一个在编辑区输入`000`+ emoji 名字（中文）就可以生成 emoji 的 vs code 插件，内置三千多个 emoji，可以让我们快速生成 emoji！

## 使用 🕹️

直接输入 000+emoji 名字，就可以生成 emoji，例如：000 龙 即可生成 🐉
